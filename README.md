# VS Code 工作目录

这个目录用于保存我们在 VS Code 工作时的协作记录、阶段报告、决策和待办。

建议使用规则：

- 每次重要讨论后，把结论追加到 `worklog/SESSION_LOG.md`
- 关键决定写入 `worklog/DECISIONS.md`
- 下一步行动写入 `worklog/NEXT_STEPS.md`
- 完整分析报告放入 `reports/`

推荐目录：

```text
VS Code/
  README.md
  worklog/
    SESSION_LOG.md
    DECISIONS.md
    NEXT_STEPS.md
  reports/
  setup/
    git-private-repo-checklist.md
    remote-tunnel-mobile-guide.md
```
