# Opula 项目备份

这个目录是从 `D:\ClaudeWork` 中整理出来的 Opula 台灯项目镜像备份。

## 备份原则

- 优先纳入：
  - 核心策略文档
  - 页面原型
  - 设计方案
  - 自动化脚本
  - 关键视频素材
  - 海报终稿与关键参考图
  - 可直接用于页面和广告的产品素材
- 暂不全量纳入：
  - `产品图片\产品实物照片\` 下的大批量原始实拍图
  - 体积大但当前不是核心交付物的中间素材

## 目录说明

- `docs/`：项目主文档、策略、品牌、专家文档
- `prototypes/`：HTML 页面原型与 Hero 方案
- `design/`：Opula 设计方案文档
- `scripts/`：图像处理与海报脚本
- `assets/`：关键图片、视频、海报、白底图
