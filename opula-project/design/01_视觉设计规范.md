# Opula灯具 - 完整设计规范

## 一、色彩系统 (Color System)

### 品牌核心色
```
主色调：
- 暖白 (Warm White): #FAF9F6 - 产品背景、页面底色
- 深灰 (Deep Gray): #333333 - 文本、导航、重点元素
- 金色 (Gold): #D4AF37 - 强调、高端感、交互反馈
```

### 扩展色板
```
中立色系：
- 极浅灰 (Off White): #FEFEFE - 卡片背景
- 浅灰 (Light Gray): #F5F5F5 - 分割线、背景纹理
- 中灰 (Medium Gray): #999999 - 次要文本、描述文本
- 浅深灰 (Lighter Deep): #666666 - 正文文本
- 黑色 (Black): #222222 - 重点强调

功能色：
- 成功绿 (Success Green): #4CAF50 - 确认、完成状态
- 警告黄 (Warning Yellow): #FFC107 - 警告、提醒
- 错误红 (Error Red): #F44336 - 错误、不可用
- 信息蓝 (Info Blue): #2196F3 - 信息提示

渐变应用：
- 金色渐变：从#D4AF37 → #E8D4A8（用于hero区背景、卡片悬停）
- 深灰渐变：从#333333 → #555555（用于强调背景）
```

### 透明度规范
```
- 100% (Opaque): #FFFFFF
- 95% (Near Opaque): rgba(51, 51, 51, 0.95)
- 85% (High): rgba(51, 51, 51, 0.85)
- 70% (Medium): rgba(51, 51, 51, 0.70)
- 50% (Half): rgba(51, 51, 51, 0.50)
- 30% (Low): rgba(51, 51, 51, 0.30)
- 15% (Very Low): rgba(51, 51, 51, 0.15)
```

## 二、字体规范 (Typography)

### 字体家族
```
主字体族：Inter
- 字重：Light(300), Regular(400), Medium(500), SemiBold(600), Bold(700)
- 后备字体：-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif

CJK字体（中文）：
- 主字体："Noto Sans SC", "Microsoft YaHei", sans-serif
- 字重：Regular(400), Medium(500), SemiBold(600), Bold(700)
```

### 字体大小规范 (Type Scale)
```
H1 (Display/Hero): 
  - 大小: 56px / 3.5rem
  - 行高: 1.2 (67.2px)
  - 字重: 700 (Bold)
  - 应用: 页面主标题、hero区标题

H2 (Large Heading):
  - 大小: 44px / 2.75rem
  - 行高: 1.25 (55px)
  - 字重: 700 (Bold)
  - 应用: 分组标题、产品列表标题

H3 (Medium Heading):
  - 大小: 36px / 2.25rem
  - 行高: 1.3 (46.8px)
  - 字重: 600 (SemiBold)
  - 应用: 产品详情页标题、功能区块标题

H4 (Small Heading):
  - 大小: 28px / 1.75rem
  - 行高: 1.4 (39.2px)
  - 字重: 600 (SemiBold)
  - 应用: 小分组标题、卡片标题

Body Large:
  - 大小: 18px / 1.125rem
  - 行高: 1.6 (28.8px)
  - 字重: 400 (Regular)
  - 应用: 导航文本、hero副标题

Body Regular:
  - 大小: 16px / 1rem
  - 行高: 1.6 (25.6px)
  - 字重: 400 (Regular)
  - 应用: 正文内容、产品描述

Body Small:
  - 大小: 14px / 0.875rem
  - 行高: 1.6 (22.4px)
  - 字重: 400 (Regular)
  - 应用: 注释文本、产品规格

Caption:
  - 大小: 12px / 0.75rem
  - 行高: 1.5 (18px)
  - 字重: 400 (Regular)
  - 应用: 图片标签、小文字说明
```

### 字体颜色应用
```
- 主文本：#333333 (Deep Gray)
- 次要文本：#666666 (Light Deep)
- 辅助文本：#999999 (Medium Gray)
- 链接文本：#D4AF37 (Gold) / 悬停: #E8D4A8
- 反色文本（在深背景上）：#FEFEFE (Near White)
- 错误文本：#F44336
```

### 字体配对示例
```
标题配对：
Heading (H1-H4) → Inter Bold/SemiBold + "Noto Sans SC" Bold/SemiBold

正文配对：
Body → Inter Regular + "Microsoft YaHei" Regular
```

## 四、响应式设计

### 4.1 断点规范
```
Mobile-First 方式

xs (Extra Small):      0px ~ 480px   - 小手机
sm (Small):           480px ~ 768px  - 大手机
md (Medium):          768px ~ 1024px - 平板
lg (Large):          1024px ~ 1280px - 笔记本
xl (Extra Large):    1280px ~ 1600px - 桌面
xxl (2XL):           1600px+        - 大屏幕
```

### 4.2 栅格系统
```
12列栅格系统，使用rem单位

基础间距单位：8px (= 0.5rem)

各断点列数与gutter：
xs:  12列, gutter:8px   → col-width = (viewport - 16px) / 12
sm:  12列, gutter:12px  → col-width = (viewport - 24px) / 12
md:  12列, gutter:16px  → col-width = (viewport - 32px) / 12
lg:  12列, gutter:24px  → col-width = (viewport - 48px) / 12
xl:  12列, gutter:24px  → col-width = (viewport - 48px) / 12

示例：
1列 = span-1
3列 = span-3 (占四分之一)
6列 = span-6 (占一半)
12列 = span-12 (占满)

列与列间距：gutter/2 on each side
```

### 4.3 外边距与内边距规范
```
垂直间距：
- 组件间：32px (2rem)  [lg及以上: 48px]
- 分组内：24px (1.5rem)
- 小元素间：16px (1rem)

水平margin：
- 容器max-width：1280px
- 水平padding：
  xs: 16px
  sm: 24px
  md: 32px
  lg: 48px
  xl: 64px
```
