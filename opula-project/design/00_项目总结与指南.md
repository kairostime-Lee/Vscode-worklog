## 🎨 核心设计决策

### 1. 色彩策略
```
主色：金色 (#D4AF37) - 代表高级、温暖、可信任
背景：暖白 (#FAF9F6) - 清洁、优雅、易读
文本：深灰 (#333333) - 高对比度、专业
```

**理由**：符合北欧高端灯具品牌的品牌调性，金色和暖白的组合营造出高级感，深灰提供了足够的可读性。

### 2. 字体选择
```
英文：Inter (现代、可读性强、精确)
中文：Noto Sans SC / Microsoft YaHei (清晰、易读)
```

**理由**：Inter 是目前最流行的Web字体，具有优秀的可读性；中文搭配系统字体确保加载速度。

### 3. 布局策略
```
桌面版：2列/3列/4列混合布局，左右对齐，充分利用屏幕
移动版：单列/双列布局，上下堆叠，内容清晰
```

**理由**：移动优先的响应式设计，确保在所有设备上的用户体验一致。

### 4. 图片处理
```
使用WEBP格式 + JPG备用
多分辨率适配 (480w/768w/1280w/2560w)
智能懒加载，优化首屏速度
```

**理由**：最大化性能，减少传输大小20-30%，同时保持高质量。

## 📞 项目支持

### 常见问题

**Q: 这个设计方案可以用于多语言吗？**
A: 可以。设计系统是语言无关的。只需要调整字体选择（中文、日文等），其他设计规范保持不变。

**Q: 如何处理不同的产品类型（吊灯、台灯等）？**
A: 设计系统的组件是通用的。可以为不同产品类型创建不同的集合页面，但核心组件保持一致。

**Q: 图片需要购买许可证吗？**
A: 推荐使用自己的产品照片（已有35张）。对于场景照，可以自己拍摄或购买库存照片（Unsplash, Pexels等）。

**Q: 如何确保网站在旧浏览器上正常运行？**
A: 使用Progressive Enhancement方法，提供降级方案。详见 04_交互与动画设计.md 的"性能优化"部分。

**Q: 需要多少预算来实现这个设计？**
A: 取决于开发方式：
- 自建团队：$20,000-40,000（3-6周开发）
- 外包代理：$30,000-60,000（包括设计+开发）
- 低代码工具：$5,000-15,000（如Shopify, Wix等）
