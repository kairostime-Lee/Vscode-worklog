## 🔑 核心要点速查

### 颜色系统（复制即用）
```css
/* 主色系 */
--color-primary: #D4AF37;     /* 金色 */
--color-primary-light: #E8D4A8; /* 亮金色 */
--color-primary-dark: #C4A838;  /* 深金色 */

/* 中立色系 */
--color-bg: #FAF9F6;     /* 暖白 */
--color-text: #333333;   /* 深灰 */
--color-text-secondary: #666666;  /* 中灰 */
--color-text-tertiary: #999999;   /* 浅灰 */

/* 功能色 */
--color-success: #4CAF50;
--color-warning: #FFC107;
--color-error: #F44336;
--color-info: #2196F3;
```

### 断点系统（复制即用）
```scss
// 移动优先断点
$xs: 320px;  // 手机
$sm: 480px;  // 大手机
$md: 768px;  // 平板
$lg: 1024px; // 笔记本
$xl: 1280px; // 桌面
$xxl: 1600px; // 超宽屏

// 媒体查询Mixin
@mixin screen($breakpoint) {
  @if $breakpoint == 'sm' {
    @media (min-width: $sm) { @content; }
  }
  @if $breakpoint == 'md' {
    @media (min-width: $md) { @content; }
  }
  // ... 其他断点
}

// 使用示例
.container {
  width: 100%;
  @include screen('md') {
    width: 80%;
  }
}
```

### 动画缓动（复制即用）
```css
/* Material Design 标准缓动 */
--easing-standard: cubic-bezier(0.4, 0, 0.2, 1);
--easing-decelerate: cubic-bezier(0, 0, 0.2, 1);
--easing-accelerate: cubic-bezier(0.4, 0, 1, 1);
--easing-sharp: cubic-bezier(0.4, 0, 0.6, 1);

/* 使用示例 */
transition: all 0.3s var(--easing-standard);
```

### 响应式字体（复制即用）
```css
/* H1 - Hero标题 */
font-size: clamp(36px, 8vw, 56px);
line-height: 1.2;
font-weight: 700;

/* Body - 正文 */
font-size: clamp(14px, 2vw, 16px);
line-height: 1.6;
font-weight: 400;

/* 说明：clamp(最小值, 首选值, 最大值) */
```
