## 二、设计系统（Design System）

### 2.1 色彩系统

```
主色（品牌色）：
├── Space Gray Dark:  #2C2826  (深空灰，主背景/CTA按钮)
├── Space Gray Mid:   #4A4E5A  (中性灰，次要文字)
├── Warm White:       #F5F2EE  (暖白，主背景/浅色区域)
├── Cream:            #EDEAE5  (奶白，交替区块背景)
└── Off White:        #FAF9F6  (极浅底色)

强调色（点缀用，不超过全页 5%）：
├── Tech Blue:        #3B82F6  (科技蓝，认证徽章/高亮数字)
├── Amber Warm:       #D4A847  (暖金，星级评分)
└── Success Green:    #22C55E  (库存/配送状态)

文字色：
├── Primary:          #1A1612  (标题，最深)
├── Secondary:        #4A3E38  (正文)
├── Tertiary:         #8A7A72  (辅助信息/标签)
└── Inverse:          #F5F2EE  (深色背景上的文字)

不推荐色（从现有代码删除）：
└── #333333 (body文字，改为 #4A3E38，更温暖)
```

### 2.2 字体系统

```
主字体：Inter（已使用）—— 保留
├── 加载优化：补充字重 300（light），当前只有 400/700

标题字体层级：
├── Display（Hero主标）：56-80px / weight 300 / line-height 1.05
├── H1（页面主标）：40-56px / weight 300+700 混用
├── H2（区块标题）：32-40px / weight 300
├── H3（功能标题）：20-22px / weight 700
├── Label（小标签）：10-11px / weight 400 / letter-spacing 0.4em / UPPERCASE

正文字体层级：
├── Body Large：17px / weight 400 / line-height 1.75
├── Body：15px / weight 400 / line-height 1.7
└── Caption：13px / weight 400 / line-height 1.6

数字展示（规格/价格）：
└── 使用 Inter 700，字号 38-52px，字距 -0.02 to -0.03em（负字距更高级）
```

### 2.3 间距系统（8px 基准网格）

```
核心间距值：
8px / 16px / 24px / 32px / 48px / 64px / 96px / 128px / 160px

区块内部 Padding（桌面端）：
├── 大区块上下 Padding: 128px（现有 features/comparison 已正确）
├── 容器水平 Padding: 64px（现有正确）
└── 卡片内 Padding: 48-56px

区块内部 Padding（移动端）：
├── 大区块上下 Padding: 72px
├── 容器水平 Padding: 24px

```

### 2.4 圆角与边框

```
风格定位：极简硬朗，拒绝圆润
├── 按钮圆角：0（方角，高级感）
├── 卡片圆角：0（方角）
├── 图片圆角：0
├── 输入框圆角：0
└── 唯一例外：信任徽章可以 4px 圆角

分隔线：
├── 颜色：#D4CFC9（现有）或 #E8E4DE
└── 粗细：1px
```

## 三、各区块设计规范

### Section 1：Hero 设计规范

**桌面端（≥1024px）：**
```
├── 全屏高度：100vh，最小 680px
├── 背景图：台灯跨越大显示器的真实场景图
│   ├── 理想尺寸：2560×1440px，WebP 格式，< 400KB
│   ├── 图片处理：降低对比度，偏暗（便于文字叠加）
│   └── 焦点：显示器上方，台灯处于中右位置
├── 渐变遮罩：左侧到中央：linear-gradient(to right, rgba(0,0,0,0.55) 0%, rgba(0,0,0,0.1) 60%, transparent 100%)
├── 文字区域（左侧 1/2）：
│   ├── 位置：left: 80px, top: 50%，transform: translateY(-50%)
│   ├── 品牌标签：OPULA TECHNOLOGY（11px / 字距 0.44em）
│   ├── 副标题：THE LAMP THAT STANDS ABOVE YOUR MONITOR（13px / 字距 0.2em）
│   ├── 主标题：（56-70px / weight 700 / 白色）
│   │   Line 1: Full Spectrum
│   │   Line 2: Ra ≥ 97 · 67cm · Hello Star
│   └── CTA 按钮组（间距 16px）：
│       ├── 主按钮：[Shop Now — $149.99]
│       │   样式：background #F5F2EE, color #1A1612, 无圆角
│       │   尺寸：padding 16px 40px, font-size 12px, letter-spacing 0.2em
│       └── 副按钮：[▶ Watch It Extend]
│           样式：border 1px solid rgba(255,255,255,0.4), 透明背景, 白色文字
└── 信任行（底部 fixed bar）：
    ├── 位置：bottom 0, 全宽, background rgba(0,0,0,0.45), backdrop-filter blur(8px)
    └── 内容：★★★★★ 4.9 · 247 Reviews  |  IEC Certified  |  Free Shipping  |  30-Day Returns
```

**移动端（< 768px）：**
```
├── 布局改为上下结构（非叠加）：
│   ├── 上部：产品图（3:4 比例，右侧留白，台灯主体居左偏中）
│   └── 下部：白色/浅色背景区块，包含所有文字和按钮
├── 主标题：32-36px
├── 按钮：全宽，高度 52px
└── 信任行：2x2 网格排列（非横排）
```

## 四、移动端专项设计规范

### 关键原则：**Facebook 流量 80% 来自手机，移动端是第一设计优先级**

```
全局移动端规则：
├── 最小触摸目标：44×44px（Apple HIG 标准）
├── CTA 按钮：全宽，高度 52-56px
├── 字体最小：14px（正文），不要小于此
├── 行间距：1.65（移动端阅读疏一点）
├── 图片：优先 aspect-ratio，不设固定高度
└── 禁止水平滚动（检查所有固定宽度元素）

移动端导航：
├── 顶部固定导航：高度 56px
│   ├── 左：汉堡菜单（三横线 SVG）
│   ├── 中：OPULA（品牌名或 Logo）
│   └── 右：购物袋图标（有商品时显示数量）
└── 底部固定 CTA 条（关键！）：
    ├── 始终显示在屏幕底部
    ├── 背景：#2C2826
    ├── 内容：[Add to Cart — $149.99]（全宽按钮）
    └── 出现时机：用户滚动超过 Hero 区域后显示
```

## 六、动效规范

```
原则：克制、有目的、不影响性能

推荐动效：
├── Hero CTA 按钮 Hover：背景色 0.18s ease（现有，保留）
├── 导航滚动时：顶部出现半透明导航条（backdrop-filter blur）
├── 数字/统计数据：Scroll into view 时数字递增动画（0 → 97，0 → 67）
├── 语音区卡片：CSS @keyframes，交错 fadeIn + translateY(20px)
├── 产品图轮播：crossfade 0.3s（不要滑动，太廉价）
└── 底部固定 CTA：首次出现 slide-up 0.2s

禁止：
├── 复杂 JS 动画库（GSAP 等，太重，影响 LCP）
├── 自动播放视频（除非 muted + loop + 用户主动点击）
└── 过多视差效果（影响移动端性能）
```

## 八、Shopify 主题推荐

```
推荐使用官方主题（省开发时间）：

首选：Dawn（免费，Shopify 官方，性能最好）
├── 优点：Core Web Vitals 优秀，零 JS 框架，全面响应式
├── 缺点：设计偏通用，需要大量自定义 CSS/Section 调整
└── 建议：购买 Dawn 增强版第三方主题如 "Prestige" 或 "Symmetry"

备选付费主题：
├── Prestige（$380，一次性）— 最接近 Dyson/高端 DTC 风格，强烈推荐
│   特点：视差效果、大图视觉、黑暗模式支持
├── Symmetry（$380）— 适合多图展示，产品摄影驱动型
└── Motion（$360）— 动效丰富，适合视频内容多的品牌

如果预算有限：
└── 免费 Dawn + 人工自定义 CSS（需要 3-5 天开发时间）
```
