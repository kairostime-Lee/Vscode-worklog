## 七、关键风险与应对

```
风险1：第一轮广告效果差（ROAS < 2）
应对：
├── 先检查落地页转化率（用 Hotjar 免费录屏）

├── 如果点击少：换创意素材（重拍视频）
└── 如果完全没流量：检查 Pixel 是否正确安装

应对：
├── 预售页面明确标注"Ships in 4-6 weeks"
├── 一旦延迟立即邮件通知所有预购客户
└── 提供延迟补偿（额外赠品或折扣码）

风险3：退款率高（客户等不及）
应对：
├── 每周发送生产进度更新邮件（增加信任感）
├── 用照片/视频展示工厂生产过程
└── 超预期发货（比承诺时间早1周更好）

风险4：竞品跟进（降价抢市场）
应对：
└── 护城河在于：专利伸缩 + Ra≥97 + 离线语音 = 三合一，竞品短期无法复制

风险5：支付宝/微信收款遇到问题
应对：
└── 初期先用 Stripe 信用卡 + PayPal 保证基本功能，华裔支付可后续加
```
